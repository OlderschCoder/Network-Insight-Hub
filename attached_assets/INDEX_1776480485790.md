## 4. Azure subscription exports (JSON bundles)

**Generator:** **`azure_visualizer.ps1`** — see **`AZURE_VISUALIZATION_CLI.md`**.

**Example folder:** **`azure-export-20260417-212136`** (timestamp varies per run).

| File pattern | Use for diagrams / analysis |
|--------------|------------------------------|
| **`20-network-vnet-list.json`** | VNet address space, subnets, DNS |
| **`25-network-nsg-list.json`** | Security rules (large) |
| **`27–29`** | VPN gateways, connections, local network gateways |
| **`40-vm-list-detail.json`**, **`50-graph-vms.json`** | VM inventory |
| **`52-graph-vnets.json`**, **`53-graph-nics.json`** | Graph-style relationship exports |
| **`99-resource-list-all.json`** | Full subscription resource list |

Re-run **`.\azure_visualizer.ps1`** after major Azure changes to refresh.
