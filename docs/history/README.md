# Project History

Maintained history of the two IT ChatGPT projects that inform this platform:

- [`network-operations-dashboard.md`](./network-operations-dashboard.md) — Network Operations Dashboard project
- [`sccc-help-desk.md`](./sccc-help-desk.md) — SCCC Help Desk project

## Why this exists

These ChatGPT project conversations live behind a private login and can't be
pulled automatically. To keep their context available to this repo (and to the
agent working on it), paste the relevant chat content into the files above.

## How to maintain

Each history file is a running, dated log. When you add content:

1. Add a new dated entry under **Timeline** (newest first), with a short title.
2. Paste the relevant chat excerpt or a summary under that entry.
3. If a decision was made, also record it under **Decisions**.
4. If something is still open, add it to **Open Items** (and remove it once done).

Keep entries factual and skimmable — summaries are better than raw dumps, but
raw excerpts are fine when detail matters. Convert relative dates ("yesterday")
to absolute dates so the log stays interpretable later.
